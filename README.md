# anim-seq

**anim-seq** is a Blender addon to import static meshes as animations.

## Features

- Import static meshes as animated sequences.

## Installation

1. Download the `anim-seq.zip` file.
2. In Blender, go to `Edit > Preferences > Add-ons > Install`.
3. Select and install the `anim-seq.zip`.
